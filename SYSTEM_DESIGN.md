# SYSTEM DESIGN

## Structure
- `HOME.md` = vault home
- `LONG_TERM_MEMORY.md` = executive summary
- `daily/` = raw journal
- `entities/` = durable references
- `decisions/` = why choices were made
- `projects/` = longer-running work
- `playbooks/` = reusable operating procedures

## Rule
Small linked notes beat one giant memory file.
