# 2026-03-14 - Adopt Memory Protocol V3

## Decision
Adopt Memory Protocol V3 for Bobby's long-term memory system.

## Why
A useful memory system needs more than storage. It needs explicit hierarchy, source-of-truth rules, consistency rules, mirror policy, and stale-memory prevention.

## Includes
- chat vs working vs daily vs durable vs executive memory layers
- Obsidian as canonical source of truth
- OpenClaw mirror as derived retrieval layer
- update-existing-note preference over duplication
- explicit promotion rules and review loops

## Expected Result
